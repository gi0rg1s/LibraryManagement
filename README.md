# 📚✨ LibraryManagement ✨📚

Your favorite cute library management system! 💕 A simple yet adorable library management system implemented in C using SQLite for data storage. The program allows users to log in, search for books by title or author, and manage their fabulous profiles! 👑

## ✨ Features
- 👤 User authentication 
- 🎀 User registration
- 📖 Add new books to the library database
- 💫 View book details
- 📤 Loan and return books
- 🔍 Search for books by title or author
- 👗 View and manage your stunning profiles

## 💖 Usage
1. 👩‍💻 Compile the program using a C compiler and link it with the SQLite library
2. 🚀 Run the compiled executable
3. ✨ Follow the on-screen prompts to log in, register, search for books, and manage your profile like the queen you are!

## 🌸 Instructions
1. 💝 Compile from windows/linux terminal:
   ```
   gcc main.c myLibrary.c -o libraryManagement -lsqlite3
   ```
